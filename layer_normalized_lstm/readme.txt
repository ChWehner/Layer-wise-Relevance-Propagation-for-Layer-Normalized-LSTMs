MOST SCRIPTS REQUIRE DATA TO RUN. THIS DATA IS DUE TO A NON-DISCLOUSURE AGGREMENT NOT ALLOWED TO BE PUBLICLY SHARED. PLEASE CONTACT THE AUTHOR (christoph.wehner@uni-bamberg.de) FOR THE DATASET.

Data Preprocessing:
1. Open your pip/conda environment
2. pip install pre_processing_requirements.txt
3. python data_preprocessing.py

For Training: 
1. Open your pip/conda environment
2. pip install requirements.txt
3. python model_five_plus.py

For Testing the Model:
1. Open your pip/conda environment
2. pip install requirements.txt
3. python predict_five_zero.py
